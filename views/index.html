Request Header Parser Microservice

Details:

GET requests to the URL https://spectacular-pest.glitch.me/api/whoami will return a JSON object containing the IP address, preferred language and system infos from the device issuing the request

Usage:

GET https://spectacular-pest.glitch.me/api/whoami

Example Output:

{"ipaddress":"159.20.14.100","language":"en-US,en;q=0.5", "software":"Mozilla/5.0 (X11; Ubuntu; Linux x86_64; rv:50.0) Gecko/20100101 Firefox/50.0"}
